Rolling Stone

Rolling Stone is a virtual Dice Game that is played between two users you select a user to challenge and place a bet with him/her. Users are ranked based on the number of points they've earned over time. so be sure to contend with the best.

How to Play Rolling Stone

After a you've posted a challenge and the challenge is accepted by the contender Each user rolls the dice and the user with highest number wins.
